What is Jquery

Jquery is a fast, small and fearure rich JavaScript library. it simplifies various tasks in web development,
Such as HTML document traversal and manipulation, event handling, animation

Why do we use Jquery

Cross browser compatibility: jQuery helps abstract away many of the differences and quirks between web browsers, making it easir to write code that works consistently across various browsers

DOM Manipulation: It provides a concise and easy to use APU for selecting and manipulating HTML elements i Document Object Model of a web page

Event handling: J query simplifies event handling by providing methods to attach and detach event handlers
